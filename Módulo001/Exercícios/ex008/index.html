Principais Formatações

Negrito / Destaque

Nessa frase temos um termo em negrito usando a tag <b> (não semântica).

Nessa frase temos um termo em destaque usando a tag <strong> (com semântica).

Itálico / Ênfase

Nessa frase temos um termo em itálico usando a tag <i> (não semântica).

Nessa frase temos um termo em ênfase usando a tag <em> (com semântica).

Recomendação do Gustavo Guanabara

Use as tags <strong> e <em> em vez de <b> e <i>.

Texto Marcado

Podemos criar também um texto marcado usando a tag <mark>.

E no outro parágrafo temos outro texto marcado no final.

Texto Grande e Pequeno

Estamos criando um texto grande e um texto pequeno nesse parágrafo. Leia o comentário no código apertando a tecla f12 no Google Chrome.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Para fazer isso, utilize a tag <del>.

Texto Inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Para fazer isso, utilize a tag <ins>

Existe também o sublinhado com tag <u> que é não semântica.

Texto Sobrescrito

Para inserir coisas do tipo x2+3. A tag utilizada é <sup>

Texto Subscrito

Para inserir coisas do tipo H2O. A tag utilizada é <sub>
